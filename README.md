## Execution
1. Execute the following to generate the default files:
   ```bash
   $ pip install -r requirements.txt
   $ python -m Bot --init-db
   ```
2. Update the generated `config.yaml` with your Discord Token and preferred Prefix
3. Run the following:
   ```bash
   $ python -m Bot
   ```
